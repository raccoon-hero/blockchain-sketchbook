⛓️ PKO's Blockchain API

Використовуйте наведені нижче кінцеві точки API для взаємодії з блокчейном. Додавайте транзакції, видобувайте блоки та переглядайте поточний блокчейн.

  • POST /transactions/new: Додати нову транзакція
  • GET /mine: Почати видобуток нового блоку
  • GET /chain: Отримати json поточного блокчейну
